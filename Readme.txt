WP Add Social Bookmarks + Animation effect is a plug-in designed to add major social bookmarks to your website. 

== Description ==

This plugin is for wordpress users, both beginner and expert.function is to distribute your articles to popular social bookmarking sites. This plugin provides social bookmarking 8 favorites such as,Facebook, twitter, reddit, digg, technorati, stumbleupon, delicious and yahoo buzz.

This plugin requires no configuration at all. simply activate and the plugin will be installed under your wordpress post articles.

image in the folder "images" in directory installation of this plugin '/wp-content/plugins/wp-add-socialbookmarks/images'. if you want to change an image, just need a little change. and can be customized to the Editor Plugin.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `WP-Add-Social-Bookmarks.zip` to the `/wp-content/plugins/` directory. You can do this using 'Upload' functionality provided in plugins section of your wordpress dashboard
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You are all set. Start bookmarking now
